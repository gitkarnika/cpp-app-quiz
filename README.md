# C++ Quiz Game 🎯

A console-based quiz game developed in C++.

## 📌 About the Project

This project is a multiple-choice quiz game designed to test basic C++ knowledge. The quiz contains two levels and provides a retest option if the required score is not achieved.

## ✨ Features

* Two levels of quiz
* Multiple-choice questions
* Accepts answers in both numbers and letters
* Supports answers like `1`, `A`, `a`, etc.
* Automatic score calculation
* Retest option for low scores
* Level progression system

## 🛠️ Technologies Used

* C++

## 🎮 How to Run

1. Clone this repository.
2. Open the C++ source file.
3. Compile the program using a C++ compiler.
4. Run the program and start the quiz.
